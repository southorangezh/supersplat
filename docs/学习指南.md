# SuperSplat 文档学习指南

本指南帮助你快速熟悉 `docs/` 目录的文档结构，并提供推荐的阅读顺序，便于循序渐进地掌握 SuperSplat 的特性和底层实现。

## 推荐阅读顺序
1. **《PlayCanvas 核心概念与 API 文档》**：了解 SuperSplat 运行所依赖的 PlayCanvas 引擎核心概念（Application、Scene、Entity、Component），为后续阅读打下基础。
2. **《PlayCanvas 渲染系统底层封装实现文档》**：深入渲染管线的封装方式，理解 SuperSplat 如何组织渲染流程和资源。
3. **《PlayCanvas 多视角摄像机系统完整指南》**：学习摄像机系统的架构与使用场景，帮助你在编辑器中配置多视角工作流。
4. **《选择工具渲染逻辑文档》**：了解选择工具的渲染与交互逻辑，掌握如何扩展或调试编辑器的选取功能。

## 学习建议
- **配合示例操作**：在阅读每篇文档时，打开 SuperSplat 编辑器实际尝试文中的操作或 API，帮助加深理解。
- **关注接口层级**：优先理解公共接口与抽象层，再深入具体实现，便于在代码中快速定位功能入口。
- **记录问题与优化点**：在学习过程中将疑问或改进想法记录下来，后续可以集中验证或讨论。
